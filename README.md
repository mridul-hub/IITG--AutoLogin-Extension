# IITG-AutoLogin-Extension
This extension will auto login you into the IITG network. No need to login everytime or keep a tab open!

### To run the chrome extension locally go to (chrome://extensions)

![address](https://user-images.githubusercontent.com/62867903/189545609-c85fb739-65ab-48bf-97d7-d3317b813388.png)

### Turn Developer mode ON, click load unpacked button and select the downloaded extension folder.

![mode](https://user-images.githubusercontent.com/62867903/189545654-64a16eb6-b621-4ed7-9ea3-d1776ba54246.png)
